# PinaToy's - フィギュア紹介サイト

ネオブライタリズムスタイルのフィギュア紹介・カタログサイトです。

## 技術スタック

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Zustand
- lucide-react

## セットアップ

```bash
yarn install
```

## 開発

```bash
yarn dev
```

## ビルド

```bash
yarn build
```

## プレビュー（本番ビルドの確認）

```bash
yarn preview
```

## リント

```bash
yarn lint
```
